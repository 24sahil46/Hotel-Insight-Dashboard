#  Hotel Insight Dashboard

##  Project Overview

The **Hotel Insight Dashboard** is a data analytics project designed to analyze hotel performance using business intelligence tools. The dashboard provides insights into key performance indicators such as revenue, occupancy rate, ADR, RevPAR, and customer ratings across different cities and hotel categories.

This project helps hotel management monitor operational performance, identify trends, and make **data-driven business decisions**.

---

##  Objectives

* Analyze hotel performance using visual analytics.
* Track important metrics such as **Revenue, RevPAR, ADR, and Occupancy Rate**.
* Compare hotel performance across **different cities and properties**.
* Provide interactive filters for better data exploration.
* Help management identify **high-performing and underperforming hotels**.

---

##  Dashboard Features

* **Total Revenue Analysis**
* **RevPAR (Revenue per Available Room)**
* **ADR (Average Daily Rate)**
* **Occupancy Percentage**
* **Realisation Percentage**
* **Average Customer Rating**
* **Revenue Distribution by Category**
* **Property Performance Comparison**
* **Weekly Revenue Trends**

Interactive filters allow users to analyze data by:

* City
* Room Class
* Hotel Category
* Time Period

---

##  Dataset Information

The dataset contains hotel operational data including:

* Hotel properties
* Booking details
* Room categories
* Revenue metrics
* Occupancy statistics
* Customer ratings

Dataset files include:

* `dim_date`
* `dim_hotels`
* `dim_rooms`
* `fact_bookings`
* `fact_aggregated_bookings`

##  System Workflow

1. Collect hotel dataset.
2. Clean and preprocess the data.
3. Load the data into Power BI.
4. Create relationships between tables.
5. Calculate key metrics (Revenue, ADR, RevPAR, Occupancy).
6. Design interactive visualizations.
7. Generate insights for decision making.
